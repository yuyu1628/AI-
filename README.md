项目简介
本项目是一个高度工程化的 AI 长篇小说创作框架，旨在通过多智能体（Multi-Agent）协作与状态机管理，实现网文小说的工业级流水线生产。
区别于传统的单次 Prompt 续写，本项目模拟了真实的人类创作团队和编辑部流转过程。从全书大纲推演、世界观设定扩充，到动态记忆提取、正文撰写，再到极其严苛的“排版+逻辑”双盲审核，最终将新产生的剧情事实反写回全局状态机。全流程自动化闭环，辅以关键节点的“人类熔断确认”机制，确保小说在达到百万字规模时，依然保持剧情逻辑连贯、人物人设不崩塌。
核心技术特性：
🧠 全局状态机与动态记忆： 独创 world_bible.json 维护角色生死、境界与核心道具归属；配合 rolling_memory.json 实现最近 5 章滑动窗口摘要与无缝情绪衔接。
🕵️‍♂️ 战前情报提取（RAG）： 撰写单章前，按需精准抽取设定集中的出场人物与相关法则，避免强行“报菜名”和设定污染。
⚖️ 双盲审判庭机制： 正文草稿需经过“排版编辑”（严格把控长短句呼吸感与去 AI 味）与“逻辑编辑”（校验设定一致性）的双重拷问，打回后由专职“精修师”进行局部定向重写。
⚙️ 七大模块流水线： 明确划分总纲规划、分卷拆解、情报组装、核心主笔、双盲审核、状态流转、结卷复盘 7 个独立模块，支持灵活接入不同能力倾向的 LLM（如逻辑极强的模型做规划，语感极佳的模型做精修）。
